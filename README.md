# Motetorio

Motetorio is a *Factorio* mod project designed to bring the six elemental planes of *Motemancer* into the Space Age! It is composed of two separate mods:
- motetorio_core: This mod serves as a standalone experience, bringing the six planes to Factorio as six distinct planets. This mod can safely be added to an existing save without breaking anything, as it changes no vanilla recipes.
- motetorio_overhaul: This is the INTENDED experience of the project, massively overhauling Factorio's gameplay. It is almost certainly incompatible with other overhaul mods.

Unfortunately, I'm not an artist or musician, so expect LOTS of janky placeholder art (all lovingly scribbled by hand, no AI here!) and royalty-free music instead of polished audio & graphics. I'll eventually figure it all out... somehow.
